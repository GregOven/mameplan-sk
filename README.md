# Mámeplán.sk

Bezplatná webová aplikácia pre pripravenosť domácností na krízové situácie. Vyvinuté na Slovensku, pre Slovensko.

## O projekte

Mámeplán.sk pomáha bežným ľuďom pripraviť sa na núdzové situácie — od výpadku elektriny cez povodne až po ozbrojený konflikt. Aplikácia Gerlach ponúka personalizované krízové plány, správu zásob, lekárničky a evakuačné zoznamy. Všetko funguje offline, bez registrácie.

Obsah článkov vychádza z oficiálnych príručiek európskych inštitúcií (WHO, BBK Nemecko, MSB Švédsko, DSNS Ukrajina, MV SR, ERC a ďalšie).
